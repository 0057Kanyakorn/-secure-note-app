# 🔒 SecureNote Application

A lightweight, full-stack web application that allows authorized users to securely create, view, and delete text notes. This project demonstrates a clear separation of concerns between the Client (Browser) and the Server (Node.js), along with secure API communication.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (DOM Manipulation, Fetch API)
- **Backend:** Node.js, Express.js
- **Architecture:** RESTful API
